Stochastic Process Model with Basketball Data

Chapter 4 Dataset

The dataset is from the Duke University Men’s Basketball SportsVu tracking data. Features were created by taking snapshots of the game every 1/25th of a second and recording the player’s location, action, team, etc. Data was collected for each season from 2013-2016; the dataset totals about 2 million observations and 72 features.
